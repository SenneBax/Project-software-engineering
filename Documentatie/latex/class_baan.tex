\doxysubsection{Detailed Description}
Klasse voor een baan in de verkeerssimulatie. 

Een \doxylink{class_baan}{Baan} object representeert een wegvak met een naam en lengte. Banen zijn de fundamentele infrastructuur elementen waarop voertuigen zich bewegen. Elke baan heeft een unieke naam binnen een verkeerssituatie en een positieve lengte in meters.

Design by Contract\+:
\begin{DoxyItemize}
\item Namen mogen niet leeg zijn
\item Lengtes moeten positief zijn
\item Object moet correct geïnitialiseerd zijn via \doxylink{class_baan_a383513a9ca564e848d4d390cabd59195}{properly\+Initialized()} 
\end{DoxyItemize}
