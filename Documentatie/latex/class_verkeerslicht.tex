\doxysubsection{Detailed Description}
Klasse die een verkeerslicht in de verkeerssimulatie voorstelt. 

Een \doxylink{class_verkeerslicht}{Verkeerslicht} regelt het verkeer op een specifieke positie op een baan. Het kan verschillende kleuren hebben (groen, oranje, rood) en schakelt automatisch volgens een cyclustijd. Slimme verkeerslichten kunnen hun gedrag aanpassen op basis van wachtende voertuigen.

Eigenschappen\+:
\begin{DoxyItemize}
\item Normale verkeerslichten\+: vaste cyclustijd
\item Slimme verkeerslichten\+: adaptieve cyclustijd gebaseerd op verkeer
\item Optionele oranje fase tussen groen en rood
\item Registratie van wachtende voertuigen voor slimme lichten 
\end{DoxyItemize}

Update de status van het verkeerslicht. Voor normale lichten schakelt het volgens vaste cyclustijd. Voor slimme lichten wordt ook rekening gehouden met wachtende voertuigen.

Normale logica\+:
\begin{DoxyItemize}
\item Met oranje\+: Groen -\/\texorpdfstring{$>$}{>} Oranje (10\% cyclus) -\/\texorpdfstring{$>$}{>} Rood -\/\texorpdfstring{$>$}{>} Groen
\item Zonder oranje\+: Groen -\/\texorpdfstring{$>$}{>} Rood -\/\texorpdfstring{$>$}{>} Groen
\end{DoxyItemize}

Slimme logica\+:
\begin{DoxyItemize}
\item Minimaal 10s, maximaal 60s per fase
\item Schakelt naar groen bij wachtende voertuigen
\item Schakelt naar rood bij geen verkeer na 20s groen
\end{DoxyItemize}
