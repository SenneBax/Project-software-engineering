\doxysubsection{Detailed Description}
Klasse voor het genereren van uitvoer van verkeerssituaties. 

De output klasse biedt verschillende methoden om verkeerssituaties te visualiseren en te exporteren. Ondersteunde formaten zijn\+:
\begin{DoxyItemize}
\item Tekstuele rapporten met statistieken
\item ASCII-\/art grafische impressies
\item XML export voor persistentie
\item HTML visualisaties voor webbrowsers 
\end{DoxyItemize}
