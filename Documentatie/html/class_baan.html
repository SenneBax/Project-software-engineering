Detailed Description

Klasse voor een baan in de verkeerssimulatie.

Een Baan object representeert een wegvak met een naam en lengte. Banen zijn de fundamentele infrastructuur elementen waarop voertuigen zich bewegen. Elke baan heeft een unieke naam binnen een verkeerssituatie en een positieve lengte in meters.

Design by Contract:

  • Namen mogen niet leeg zijn
  • Lengtes moeten positief zijn
  • Object moet correct geïnitialiseerd zijn via properlyInitialized()
