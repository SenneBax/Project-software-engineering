Detailed Description

Klasse voor het genereren van uitvoer van verkeerssituaties.

De output klasse biedt verschillende methoden om verkeerssituaties te visualiseren en te exporteren. Ondersteunde formaten zijn:

  • Tekstuele rapporten met statistieken
  • ASCII-art grafische impressies
  • XML export voor persistentie
  • HTML visualisaties voor webbrowsers
