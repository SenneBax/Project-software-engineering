Detailed Description

Klasse die een verkeerssituatie voorstelt.

De VerkeersSituatie klasse is de centrale container voor alle elementen van een verkeerssimulatie. Het beheert de relaties tussen verschillende verkeerscomponenten en zorgt ervoor dat de gehele situatie consistent blijft gedurende de simulatie.

Precondition
properlyInitialized() == true
Postcondition

Bij true: alle voertuigen/lichten/etc zijn op geldige banen met geldige posities

Controleert of:

  • Er minimaal één baan bestaat
  • Alle voertuigen op bestaande banen staan binnen baanlengte

  • Alle generators verwijzen naar bestaande banen
  • Alle bushaltes op bestaande banen staan binnen baanlengte

Returns
